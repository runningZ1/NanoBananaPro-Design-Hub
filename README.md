# NanoBananaPro Design Hub

Monorepo layout for the project.

## Structure

- `apps/platform` — original application (configs, scripts, docs, supabase, etc.).
- `apps/web` — Next.js frontend.

## Getting Started

### Frontend (apps/web)
- `cd apps/web`
- `npm install`
- `npm run dev`

### Platform (apps/platform)
- `cd apps/platform`
- Follow the existing scripts/config in that directory (`README.md`, `scripts/`, `docs/`, etc.).
