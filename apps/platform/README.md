# AI图片编辑专家小程序

这是一个基于Taro + React + TypeScript的微信小程序，提供20+专业AI图片处理功能。

---

## 项目概述

AI图片编辑专家是一款专业的AI图片处理工具，集成了20种强大的图片编辑功能，包括：
- AI图片转角度
- 物体6视图
- Nano Banana核心编辑
- AI文字生图
- 单图智能修改
- 多图编辑/合并
- 高清修复放大
- 产品图融合
- 材质替换
- 建筑立面迁移
- 家具融入空间
- 拷图/局部抠图
- 软装拼贴渲染
- 物体替换/添加
- 相似图生成
- 背景图生成
- 去除水印
- 清空房间家具
- AI图生视频
- 图片重新打光

## 设计系统

### 配色方案
- **主色调**: 科技蓝 (#4A90E2 / hsl(207 90% 54%))
- **背景色**: 白色 (#FFFFFF)
- **卡片背景**: 白色带轻微阴影
- **文字颜色**: 深灰色 (hsl(215 16% 20%))
- **辅助色**: 浅蓝灰 (hsl(210 40% 96%))

### 视觉特点
- 8px圆角设计
- 轻微阴影效果增强层次感
- 线性图标风格
- 网格式卡片布局
- 响应式触摸反馈

## 功能特性

### 首页
- 网格式布局展示20个AI功能
- 每个功能卡片包含图标、名称和描述
- 支持点击进入功能详情页
- "new"标签标识新功能

### 功能详情页
- 图片上传功能（支持相册和相机）
- 预制提示词选择器（每个功能5个预设提示词）
- 示例效果图展示
- 开始处理按钮
- 友好的用户提示

### 物体6视图（完整实现）✨
- ✅ 专用功能页面，完整的交互流程
- ✅ 支持上传图片或使用示例图片
- ✅ 一键生成物体的6个视角（前、后、左、右、上、下）
- ✅ **真实的6视图效果**：使用AI搜索的6张不同角度沙发图片
- ✅ 网格布局展示6个视图结果，每个视图带有视角标签
- ✅ 支持重新生成和长按保存图片
- ✅ 智能提示：示例图片可完整演示，用户上传图片会提示需要API集成
- ✅ 使用真实示例图片（现代沙发的6个不同视角）

## 安装和使用

```bash
# 安装依赖
pnpm install

# 代码检查
pnpm run lint

# 开发模式（微信小程序）
pnpm run dev:weapp

# 开发模式（H5）
pnpm run dev:h5

# 构建生产版本
pnpm run build:weapp
```

## 技术栈

- **框架**: Taro 3.x
- **UI库**: React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Material Design Icons (mdi)
- **包管理**: pnpm

## 路由配置

所有路由在 `src/app.config.ts` 中定义：
- `/pages/home/index` - 首页（功能列表）
- `/pages/feature/index` - 功能详情页（动态路由，通过id参数区分不同功能）
- `/pages/object-6-views/index` - 物体6视图专用页面（完整实现）
